A video download app tailored for YouTube Shorts and Instagram Reels, providing users with a seamless solution to save, share and enjoy their favorite short-form videos offline. 
With a user-friendly interface, the app allows quick and easy downloads, enhancing the viewing experience on the go. 

How to use project -
First clone in vs code. then hit yarn install or npm install then this is a expo project so run expo start and on mobile download expo app and then scan qr code from vs code terminal and enjoy.
